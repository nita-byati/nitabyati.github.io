Project Overview

The Denver Zoo's current ticket purchasing system created friction for visitors, leading to long lines and frustration. This project focused on redesigning the digital ticket purchasing experience to be more intuitive and efficient for families and zoo visitors.

The Challenge

Visitors struggled with the existing ticket system due to confusing pricing tiers, unclear visit planning information, and a lengthy checkout process. The zoo needed a solution that would reduce wait times and improve the overall visitor experience from the first touchpoint.

Current System Analysis

Research & Discovery

I conducted user interviews with zoo visitors and analyzed the current system to identify key pain points:

  • Confusing ticket pricing and membership options
  • Difficulty understanding what's included with admission
  • Lack of clear visit planning information
  • Complex checkout process for families with children
User Journey Mapping

Design Solution

I redesigned the ticket purchasing flow with clear information hierarchy, simplified pricing presentation, and streamlined checkout. The new design prioritizes family-friendly navigation and transparent pricing.

Key Learnings

This project highlighted the importance of understanding the specific context of family visits and time-sensitive decisions. The biggest improvements came from simplifying information presentation and reducing cognitive load during the purchasing process.

Testing with actual families revealed insights about decision-making patterns that weren't apparent in individual user interviews.
